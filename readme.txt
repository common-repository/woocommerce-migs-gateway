Woocommerce Migs allows securely payment using your Credit Card.

== Description ==
This is Woocommerce Migs Payment Gateway extension which helps to make payment securely using mastercards.

== Installation ==
1.Download the Netaxept Payment Gateway plugin zip file and extract to the  `/wp-content/plugins/` directory.
2.Activate the plugin through the 'Plugins' menu in WordPress Admin.

== Frequently Asked Questions ==
= Is it woo commerce supported? =
Yes. It is a woo commerce plugin.

= Do I need to install woocommerce plugin? =
Yes, you should install woocommerce plugin first.


== Screenshots ==
1. WooCommerce Backend Settings "Checkout" page.

4. This is Front-End Checkout Page where you have to select Credit Card-Mastercard worldwide.
